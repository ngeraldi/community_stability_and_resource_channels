## Overview

This repository contains all R code and data needed to reproduce the statistical analyses and figures in the manuscript. The experiment used 144 replicate marine mesocosms in a fully-crossed factorial design to test how trophic complexity — manipulated through four biotic factors — affects ecosystem functioning and temporal stability.

**Experimental factors:**
- **Crab** (predator): Absent / Present
- **Amphipods** (mesograzers): Absent / Present
- **Ulva** (green macroalgae): Absent / Present
- **Kelp** (structural/habitat): Absent / Intact piece / Ground up

**Response variables:**
- Benthic microalgal biomass (BenthoTorch fluorometry; 4 time points)
- Oxygen metabolism: Gross Primary Production (GPP), Net Community Production (NCP), Respiration
- Dissolved nutrients: nitrate, ammonium, phosphate
- Organism dry biomass: crab, amphipods, Ulva, kelp, detritus
- Mesofauna community on artificial pot-scrubber substrates: abundance, richness, diversity (Simpson), species composition
- Temporal stability: detrended variance of each response over the 4 sampling periods

## How to run

1. **Install required R packages:**
   ```r
   install.packages(c("vegan", "car", "reshape", "psych",
                      "lme4", "MASS", "plyr", "lmerTest", "mvabund"))
   ```

2. **Open** `R/analysis.R` in RStudio (or any R environment).

3. **Set your working directory** to the `R/` folder before running — the script uses relative paths (`../data/`, `../figures/`).

4. **Source the script** (RStudio: Ctrl+Shift+S / Cmd+Shift+S, or `source("R/analysis.R")`).

Figures will be saved as PDF files in the `figures/` directory.

## Data files description

| File | Contents |
|------|----------|
| `experiment_design.csv` | Mesocosm ID, treatment assignments (Crab/Pods/Ulva/Kelp), water flow rates, Ectocarpus cover at first sampling, BenthoTorch readings at sampling 1 |
| `oxygen_metabolism.csv` | Raw dissolved oxygen readings from light and dark incubations across all 4 sampling periods |
| `nutrients.csv` | Dissolved nitrate, ammonium, and phosphate concentrations per mesocosm |
| `nitrate_initial_final.csv` | Nitrate concentrations at experiment start and end |
| `benthic_chl_sample2.csv` | BenthoTorch pigment readings at sampling period 2 |
| `benthic_chl_sample3.csv` | BenthoTorch pigment readings at sampling period 3 |
| `benthic_chl_sample4.csv` | BenthoTorch pigment readings at sampling period 4 |
| `dry_weights.csv` | Dry weight biomass for all organism types (Ulva, amphipods, kelp, detritus, crab) |
| `crab_weights.csv` | Crab wet weights at start and end of experiment |
| `amphipod_counts.csv` | Amphipod counts and size measurements |
| `algae_cover.csv` | Ectocarpus percent cover across sampling periods |
| `mesofauna_community_counts.csv` | Mesofauna taxon abundance counts on pot-scrubber substrates across 4 time points |
| `mesofauna_scrubber_lengths.csv` | Length of each pot-scrubber substrate (for biomass standardisation) |
| `mesofauna_size_measurements.csv` | Individual body size measurements for each taxon |
| `length_weight_conversions.csv` | Length-weight regression parameters (slope, intercept) by taxon |
